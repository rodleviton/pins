# pins

A demo site example integrating with the [Pinterest](https://developers.pinterest.com) developer API and [Firebase](https://firebase.google.com).

### Installation

Prerequisites: [Node.js](https://nodejs.org/en/) (>=4.x, 6.x preferred).

``` bash
$ npm install
```

### Usage

``` bash
$ npm start
```
